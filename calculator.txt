-------------------Python Calculator------------
       Welcome to the Python Calculator project!
 This calculator is a command-line application built using Python. 
It provides basic arithmetic operations such as addition, subtraction, multiplication, and division.
---------------Features----------------
Perform basic arithmetic operations:
     	*addition
	* subtraction
	* multiplication
	* division

-----------User-friendly command-line interface.-------

  Error handling for invalid inputs and division by zero.
-----------------installation---------------------
*To run this project, you'll need Python installed on your machine.
* You can download it from the official Python website.
-------------------Usage--------------------------
   Once you run the calculator, you will be prompted to enter two numbers and choose an operation.
   The result will be displayed in the terminal.
